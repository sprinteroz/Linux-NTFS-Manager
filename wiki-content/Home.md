## ğŸš€ Quick Start

### Installation (3 steps)

1. **Clone the repository:**
   ```bash
   git clone https://github.com/sprinteroz/Linux-NTFS-Manager.git
   cd Linux-NTFS-Manager
   ```

2. **Run the installer:**
   ```bash
   ./ntfs-manager-production/install.sh
   ```

3. **Launch the application:**
   ```bash
   ntfs-manager
   ```
