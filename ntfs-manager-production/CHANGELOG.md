# NTFS Manager - Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- Production-ready package structure
- Comprehensive Nautilus integration
- Enhanced NTFS management capabilities
- Automated installation scripts
- Complete dependency management

### Changed
- Modularized backend architecture
- Improved error handling and logging
- Enhanced user interface design

### Fixed
- Drive detection issues
- Permission handling
- Memory leaks in long-running operations

## [3.0.0] - 2025-11-03

### Added
- **Nautilus Extension Integration**
  - Right-click context menu for NTFS drives
  - Properties page integration
  - Quick mount/unmount operations
  - Drive health monitoring

- **Enhanced Backend Architecture**
  - Modular drive management system
  - Advanced NTFS properties handling
  - GParted integration for partition management
  - Comprehensive logging system

- **Production Package Structure**
  - Standalone GUI application
  - Nautilus extension
  - Complete installation automation
  - Dependency management

- **System Integration**
  - Desktop file associations
  - System tray support
  - Auto-mount configuration
  - Update management system

### Changed
- Complete rewrite of backend modules
- Improved GTK3 interface
- Enhanced error handling
- Better performance for large drive operations

### Fixed
- Drive enumeration on systems with multiple controllers
- NTFS permission handling
- Memory usage during drive scans
- UI responsiveness during long operations

### Security
- Added privilege escalation for system operations
- Improved permission checking
- Safer temporary file handling

## [2.1.0] - 2025-10-15

### Added
- Basic NTFS drive detection
- Simple mount/unmount operations
- GTK3-based user interface
- Basic logging functionality

### Fixed
- Initial release stability issues
- Basic error handling

## [2.0.0] - 2025-09-20

### Added
- Project foundation
- Basic drive management
- Installation framework

### Changed
- Complete restructure from v1.x

## [1.x.x] - Legacy Versions

### Legacy Features

## Migration Notes

### From 2.x to 3.0.0
- Backup existing configurations
- Run new installation script
- Reconfigure any custom settings
- Test Nautilus integration

### From 1.x to 3.0.0
- Complete reinstall required
- All previous configurations incompatible
- New dependency requirements
